About Review Issues

Review issues are flagged by AI analysis or screenshot comparisons during test runs. Each issue needs human review to determine if it's a real problem.

Statuses:

  • Pending — awaiting review
  • Confirmed — reviewed, this is a real issue that needs fixing
  • Resolved — the issue has been fixed
  • Dismissed — not a real issue. Dismissing also creates an auto-dismiss rule so the same issue won't be flagged on future runs.

Use the Filters to narrow by status, source type, assessment, or owner. Use Bulk actions at the bottom to update multiple items at once.
