About Items

What are Items?

Items are individual test targets within an assessment, such as a specific question, stimulus, or interactive element. Each item has an ID, title, category, tier, and may be linked to an assessment and environment.

Managing Items

  • Use the search bar to find items by ID or title.
  • Click a column header to sort the table.
  • Select rows using the checkboxes, choose a bulk action, and click Apply to act on multiple items at once.
  • Click an item ID to open its detail page, where you can edit its details or delete it.

Creating Test Scripts

Test scripts are managed from the item detail page. Click on any item to view its linked scripts. Use the New Test Script button in the Test Scripts section to create a new script for that item.
