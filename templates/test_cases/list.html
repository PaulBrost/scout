About Test Plans

Test plans are automated Playwright scripts that run against assessment environments. Each test plan targets a specific environment and optionally an assessment or item.

Types of tests:

  • Baseline Screenshots — capture reference screenshots for every screen in an assessment
  • Visual Regression — compare current screenshots against baselines to detect layout changes
  • Spelling & Grammar — extract on-screen text and run AI analysis for language errors
  • QC Checklist — automatically verify interactive elements (radio buttons, checkboxes, dropdowns, etc.)

Creating a test plan:

  1. Click New Test Plan and select an environment, then an assessment or item.
  2. The AI-powered editor opens where you can describe what you want to test in plain language.
  3. Review the generated script, save, and run it directly from the editor.

Running: Test plans can be run individually from the editor, or grouped into Suites for batch execution and scheduling.
