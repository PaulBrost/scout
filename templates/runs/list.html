About Test Results

This page shows the results of all test plan and suite executions. Each row represents a single run — either a standalone test plan or a suite of test plans executed together.

Statuses:

  • Running — the test is currently executing
  • Completed — the test finished successfully (may still have flagged issues)
  • Failed — the test encountered an error or timed out

Running Tests:

To run a test, go to Test Plans and open a test plan in the editor, or go to Suites and run an entire suite. Results will appear here automatically.
