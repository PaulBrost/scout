About Assessments

What are Assessments?

Assessments represent test sessions or forms that group related items together. Each assessment defines a scope for testing, such as a specific form, module, or content area. Assessments have a subject, grade level, year, and are linked to an environment.

Managing Assessments

  • Use the search bar to find assessments by name.
  • Click a column header to sort the table.
  • Select rows using the checkboxes, choose a bulk action, and click Apply to act on multiple assessments at once.
  • Click an assessment name to open its detail page, where you can edit its details or delete it.

Creating Test Scripts

Test scripts are managed from the assessment detail page. Click on any assessment to view its items and associated scripts. Use the New Test Script button in the Test Scripts section to create a new script linked to that assessment.
