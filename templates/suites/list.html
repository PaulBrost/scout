About Test Suites

A test suite groups multiple test plans together so they can be run as a batch.

Key features:

  • Batch execution — run all test plans in a suite with a single click
  • Scheduling — set suites to run automatically on a recurring schedule
  • Per-plan config — each test plan in a suite can have its own browser and viewport

Getting started: Click the + button to create a new suite, select an environment, add test plans, and save.

Results from suite runs appear on the Test Results page.
